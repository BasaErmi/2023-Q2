# Funny JSON Explorer 设计文档

Funny JSON Explorer（**FJE**），是一个JSON文件可视化的命令行界面小工具：

- 使用python编写完成，支持所有复杂json格式的可视化。
- 工具内置了两种（树形和矩形）可视化风格的实现
- 工具自带十种
